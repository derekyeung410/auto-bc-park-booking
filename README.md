## Auto Reservation for BC Parks Campsite ⛰️

A script for automatic detection and reservation for available campsites on the BC Parks reservation website once bookings open.

## 🧠 Technologies

- `Tampermonkey`
- `JavaScript`
- `HTML`

## ✏️ Installation & Preperation

1. Download and install Tampermonkey on Chrome Web Store
   - `Extensions > Manage extensions`

1. Setting for extension
   - `Tampermonkey > Details > Enable 'Allow User Scripts'`
   - `Pin Tampermonkey `
   
3. Set up for the script:
   - `Dashboard > Create a new script`
   - `Copy the script, save and enable it`

4. Browse the BC parks website https://camping.bcparks.ca/
   - `Search and select for desired date range and campsite details`
   - `Open as LIST VIEW`
